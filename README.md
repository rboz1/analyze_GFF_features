## Getting Started

### Prerequisites
* bash

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rboz1/analyze_GFF_features.git
   
2. Modify ability to run script
   ```
   chmod u+x analyze_GFF_features.sh
4. Run the script (with desired human chromosome number 1-22, MT, X, or Y as first argument) - here we run it with chromosome 1
   ```
   ./analyze_GFF_features.sh 1
